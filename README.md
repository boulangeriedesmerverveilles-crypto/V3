# BDM — Feuille de temps (PWA)

Lien prod (GitHub Pages) :
https://boulangeriedesmerveilles-crypto.github.io/BDM/

Fonctionnalités :
- Mode salarié sans code
- Mode manager (code BDM2026)
- Mercredi/jeudi grisés
- Absences : CA / AM / AI / CFA (CFA = 7h)
- Arrondi au quart d’heure
- Retards visibles seulement manager
- Commentaires 250 caractères
- Signature mensuelle + export PDF via impression iOS
- Offline (Service Worker)
